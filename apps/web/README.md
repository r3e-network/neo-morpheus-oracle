# Morpheus Web

This is the Vercel-ready frontend and API proxy for Morpheus Oracle.

## Exposed API routes

- `/api/oracle/public-key`
- `/api/oracle/query`
- `/api/oracle/smart-fetch`
- `/api/compute/functions`
- `/api/compute/execute`
- `/api/feeds/[symbol]`
- `/api/sign/payload`
- `/api/relay/transaction`
